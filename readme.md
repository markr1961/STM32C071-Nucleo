## STM32C071 Nucleo-64 Project

A test project built around the [NUCLEO-C071RB](https://www.st.com/en/evaluation-tools/nucleo-c071rb.html) dev board from ST Microelectronics.

### CPU
#### [STM32C071RB](https://www.st.com/en/microcontrollers-microprocessors/stm32c071rb.html)
Arm Cortex-M0+ MCU 128K Flash, 24K RAM, 48 MHz CPU, 2x USART, timers, ADC, comm. I/F, USB FS, 2-3.6V

### Nucleo-64
#### I/O and Connectivity
Up to 51 I/O lines accessible via ST morpho headers and Arduino Uno V3 connectors, enabling easy expansion with shields and modules.

#### USB Support
Integrated USB connectivity for both host and device modes without requiring an external crystal 

#### Debugging and Programming
Built-in ST-LINK/V2-1 programmer/debugger; supports programming and debugging without external hardware. Can also interface with external MCUs via MIPI-10 (STDC14) connector 

#### Additional Components
- User button 
- RESET button
- (2) LEDs, 1 shared with Arduino interface
- 32.768 kHz oscillator for RTC
